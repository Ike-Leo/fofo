## Design Context

**Aesthetic Direction**: Twitter-inspired theme with distinctive typography
- Primary color: #1D9BF0 (Twitter blue in OKLCH: 0.6723 0.1606 244.9955)
- Fonts: Open Sans (UI), Georgia (editorial), Menlo (code)
- Radius: 1.3rem for rounded aesthetic
- Design philosophy: Bold, refined, avoid generic AI aesthetics

**Key Design Principles** (from frontend-design skill):
- Avoid generic fonts (Inter, Roboto, Arial, system fonts)
- Avoid purple gradient cliches
- Use distinctive typography with character
- Bold primary color with sharp accents
- High-impact animations for key moments (page load, hover states)
- Generous negative space OR controlled density
- CSS-only animations preferred

**Theme Colors**:
- Light mode: Clean whites, subtle grays, blue primary
- Dark mode: Pure black background (#000000), reduced saturation
- OKLCH color space for consistent color manipulation

## Key Learnings

1. **OKLCH Color Space**: Using OKLCH for theme colors provides consistent color manipulation and better perceptual uniformity
2. **Semantic CSS Variables**: Critical for theme consistency - use bg-background, text-foreground, border-input instead of hard-coded colors
3. **Distinctive Typography**: Open Sans provides better character than generic Inter/Roboto
4. **Rounded Corners**: 1.3rem (rounded-xl) creates a more modern, friendly aesthetic
5. **Motion Design**: CSS-only animations perform better and don't require JS
6. **Theme Transitions**: 0.3s ease provides smooth transitions without feeling sluggish
